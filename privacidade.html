Política de Privacidade — Postagem Automática

Última atualização: 04/12/2025

Esta Política de Privacidade descreve como o Postagem Automática coleta, utiliza e protege os dados fornecidos ou autorizados pelo usuário por meio da API do Instagram/Meta.

1. Coleta de Dados

Coletamos apenas dados autorizados explicitamente pelo usuário ao conectar sua conta Instagram/Facebook, tais como:

  • Informações básicas da conta
  • Lista de mídias e conteúdos
  • Métricas e insights permitidos
  • Permissão para criar publicações

2. Uso dos Dados

Os dados coletados são usados exclusivamente para possibilitar as funcionalidades do aplicativo, como:

  • Agendar e publicar conteúdo
  • Gerenciar posts
  • Exibir métricas de desempenho

Não utilizamos os dados para fins comerciais ou externos.

3. Compartilhamento de Dados

Os dados não são vendidos, compartilhados ou cedidos a terceiros, exceto quando exigido por lei.

4. Armazenamento e Segurança

Se algum dado for salvo temporariamente, será armazenado com medidas razoáveis de segurança. Nenhum sistema é 100% seguro, mas adotamos boas práticas para proteger as informações do usuário.

5. Dados do Instagram/Meta

Todos os dados são tratados de acordo com os Termos da API do Instagram/Meta. O usuário pode revogar permissões a qualquer momento nas configurações do Facebook/Instagram.

6. Direitos do Usuário

O usuário pode solicitar revisão, remoção ou esclarecimento sobre o uso de suas informações entrando em contato conosco.

7. Alterações na Política de Privacidade

Podemos atualizar esta Política ocasionalmente. A nova versão substituirá a anterior e estará disponível nesta mesma página.
